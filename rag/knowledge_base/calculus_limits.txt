L'Hopital's Rule (for 0/0 or ‚ąě/‚ąě forms):
lim f(x)/g(x) = lim f'(x)/g'(x), if the latter limit exists
Can apply repeatedly if result is still indeterminate.

Algebraic tricks for limits:
- Rationalization: multiply by conjugate for sqrt terms
- Factoring: cancel common factors
- Substitution: replace x with a+h, h‚Üí0
- Series expansion:

Sandwich (Squeeze) Theorem:
If g(x) <= f(x) <= h(x) near a, and lim g(x) = lim h(x) = L, then lim f(x) = L
Useful for: lim x*sin(1/x) = 0 as x‚Üí0

Continuity:
f is continuous at a if: f(a) is defined, lim_{x‚Üía} f(x) exists, and equals f(a)
Left limit = Right limit = function value
