QUADRATIC EQUATIONS

Standard form: ax^2 + bx + c = 0
Quadratic formula: x = (-b ± sqrt(b^2 - 4ac)) / (2a)

Discriminant D = b^2 - 4ac
- D > 0: two distinct real roots
- D = 0: one repeated real root (x = -b/2a)
- D < 0: two complex conjugate roots

Sum of roots: alpha + beta = -b/a
Product of roots: alpha * beta = c/a
Difference of roots: |alpha - beta| = sqrt(D)/|a|

Forming quadratic from roots:
x^2 - (sum of roots)x + (product of roots) = 0

Nature of roots:
- Both roots positive: D >= 0, sum > 0, product > 0
- Both roots negative: D >= 0, sum < 0, product > 0
- Roots of opposite sign: product < 0
- One root zero: c = 0

Completing the square:
ax^2 + bx + c = a(x + b/2a)^2 + (c - b^2/4a)

Vertex of parabola y = ax^2 + bx + c at x = -b/2a, y = c - b^2/4a

Maximum value of ax^2+bx+c when a < 0: (4ac - b^2) / (4a)
Minimum value when a > 0: (4ac - b^2) / (4a)

Condition for a root between p and q (p < q):
f(p) and f(q) must have opposite signs (Intermediate Value Theorem)
If both roots must lie in (p, q): D >= 0, f(p) > 0 when a > 0, f(q) > 0 when a > 0, p < -b/2a < q

Range of expression (ax^2 + bx + c) / (dx^2 + ex + f):
Let y = expression, rearrange to quadratic in x, require D >= 0 for real x.

Common JEE applications:
1. Finding integer values of k for which roots are real
2. Quadratic expressions always positive: a > 0 and D < 0
3. Common roots between two quadratics: if quadratics share root r, substitute r in both
4. Biquadratic equations: substitute t = x^2 to get quadratic in t
