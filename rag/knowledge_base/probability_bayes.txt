BAYES THEOREM AND TOTAL PROBABILITY

Bayes Theorem (detailed):
P(Hi | A) = P(A | Hi) * P(Hi) / sum_{j=1}^{n} P(A | Hj) * P(Hj)

Where:
- H1, H2, ..., Hn are mutually exclusive, exhaustive hypotheses (priors)
- A is the observed event
- P(Hi) = prior probability of hypothesis Hi
- P(A | Hi) = likelihood of observing A given Hi is true
- P(Hi | A) = posterior probability (updated belief)

Step-by-step Bayes procedure:
1. Identify all hypotheses Hi that partition the sample space
2. Assign prior probabilities P(Hi) - these must sum to 1
3. Determine likelihoods P(A | Hi) for each hypothesis
4. Compute P(A) using total probability: P(A) = sum P(A|Hi)*P(Hi)
5. Apply Bayes: P(Hi|A) = P(A|Hi)*P(Hi) / P(A)

Classic Example - Disease Testing:
Disease prevalence (prior): P(D) = 0.01, P(D') = 0.99
Test sensitivity: P(+|D) = 0.95 (true positive rate)
Test specificity: P(-|D') = 0.90, so P(+|D') = 0.10 (false positive rate)
P(+) = P(+|D)*P(D) + P(+|D') * P(D') = 0.95*0.01 + 0.10*0.99 = 0.1085
P(D|+) = 0.95*0.01 / 0.1085 ≈ 0.0876 (only ~8.76% chance of disease given positive test)

Urn Problems (common JEE type):
Two urns: urn 1 has r1 red, w1 white; urn 2 has r2 red, w2 white
Ball drawn randomly (equal chance of either urn), ball is red.
P(from urn 1 | red) = P(red | urn 1) * P(urn 1) / P(red)
P(red | urn 1) = r1/(r1+w1); P(urn 1) = 1/2
P(red) = (1/2)*r1/(r1+w1) + (1/2)*r2/(r2+w2)

Common mistake: confusing P(A|B) with P(B|A). These are different!
P(has disease | positive test) is NOT the same as P(positive test | has disease).
